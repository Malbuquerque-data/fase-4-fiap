# Fase 4 FIAP

This project is a machine learning project structured to facilitate data analysis, model training, and results visualization.

## Directory Structure

- `data/` - Contains datasets and bases used in this project.
- `notebooks/` - Contains Jupyter notebooks for data analysis and model development.
- `models/` - Holds the trained models for deployment or further evaluation.
- `src/` - Contains source code and scripts for data processing and model training.
- `results/` - Stores outputs and results from the analyses and model evaluations.

## Getting Started

To get started with this project, clone the repository and install the required dependencies. Follow the guidelines in the respective directories for usage.
